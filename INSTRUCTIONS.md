# Instructions pour finaliser l'application

## Étapes restantes :

### 1. Copier les images nécessaires

Vous devez copier manuellement les images suivantes dans les dossiers appropriés :

#### Image principale :
- **Source** : L'image du globe que vous avez fournie
- **Destination** : `assets/images/globe.png`

#### Drapeaux des pays :
Copiez ou téléchargez les drapeaux suivants dans `assets/images/flags/` :
- `france.png`
- `tunisie.png` (vous avez déjà cette image)
- `bresil.png`
- `italie.png`
- `canada.png`
- `australie.png`
- `allemagne.png`
- `espagne.png`
- `afrique_du_sud.png`

**Note** : Vous pouvez télécharger des drapeaux depuis https://www.countryflags.com

### 2. Initialiser le projet Flutter

Ouvrez un terminal dans le dossier `atlas_geo_azyyz` et exécutez :

```bash
flutter pub get
```

### 3. Exécuter l'application

```bash
flutter run
```
